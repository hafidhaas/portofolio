Pemetaan Sebaran Populasi (Random Forest)

§ 01

Eksplorasi yang saya tulis saat magang di UNFPA: memodelkan sebaran populasi menggunakan Random Forest dari variabel spasial — kedekatan jalan, fasilitas, elevasi (DEM), dan tutupan lahan (LULC). Tujuannya memperhalus estimasi populasi agar lebih sesuai dengan pola permukiman nyata (pendekatan dasymetric).

Cuplikan Kode
rf_analysis_weighted.py 
# Model dua-segmen: area terbangun vs lainnya
import numpy as np, pandas as pd
from sklearn.model_selection import train_test_split
from sklearn.ensemble import RandomForestRegressor
from sklearn.metrics import r2_score, mean_absolute_error

data  = pd.read_csv("RF_Point.csv")
X     = pd.get_dummies(data[["Jalan","Facility","DEM","LULC"]], columns=["LULC"])
y     = data["Population"]
built = data["LULC"] == 7  # 7 = area terbangun

def fit_eval(Xg, yg, **params):
    # split dulu -> latih -> evaluasi di data uji (bukan data latih)
    Xtr, Xte, ytr, yte = train_test_split(Xg, yg, test_size=0.3, random_state=42)
    m = RandomForestRegressor(random_state=42, **params).fit(Xtr, ytr)
    pred = m.predict(Xte)
    print("R2", round(r2_score(yte, pred), 3),
          "MAE", round(mean_absolute_error(yte, pred), 1))
    return m

# dua model terpisah untuk pola kepadatan yang berbeda
rf_built = fit_eval(X[built],  y[built],  n_estimators=300, max_depth=20)
rf_other = fit_eval(X[~built], y[~built], n_estimators=200, max_depth=15)

Metrik dihitung pada data uji (held-out), bukan data latih — dengan rincian akurasi per kelas tutupan lahan. Kode lengkap beserta README & catatan keterbatasan ada di repo.
